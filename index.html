Bagaimana kalimat tersebut ditulis?
Beberapa kata pada teks di atas memiliki penampilan yang berbeda dari teks biasanya. Hal tersebut dapat dilakukan dengan menggunakan tag <i> untuk miring, tag <b> untuk tebal, dan tag <u> untuk garis bawah, serta tag <blockquote> untuk
baris kutipan khusus.
Ada tambahan untuk membuat simbol '<' dan '>' agar dapat ditampilkan bukan dalam bentuk tag, yaitu dengan menulis '&-l-t-;' dan '&-g-t-;' secara bersambung.

Nomor Dwah

Beberapa orang tidak mengetahui cara menuliskan O2 atau Cm2 sebagaimana seharusnya ditulis. Sehingga sekarang saya akan menunjukkan bagaimana caranya menulis hal semacam itu dalam script HTML.

Penulisan angka ataupun huruf dengan format di atas atau di bawah tulisan utama dalam HTML dapat menggunakan tag <sub> untuk teks subskrip dan <sup> untuk teks superskrip. Contoh:
H2O, E=mc2.
